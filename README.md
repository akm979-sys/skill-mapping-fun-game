Inspired by the Saptarishi, this fun game helps children discover their unique skills across areas like logic, creativity, and social ability. It promotes self-awareness, confidence, and balanced growth through playful assessment and personalized insights.
🎮 Skill Mapping Fun Game
An engaging, school-ready web game that helps children discover their unique strengths through a fun questionnaire and instant visual report. Inspired by the Saptarishi, it blends timeless insight with modern skill mapping.

🌟 What Makes It Special
    • 🎯 70-question interactive assessment (0–3 scale)
    • 🧠 7 intelligence areas mapped with clear scores
    • 📊 Visual report card with progress bars
    • 🏆 Top 3 skills ranking for each student
    • 📈 Improvement suggestion for weakest area
    • 📤 Smart sharing (WhatsApp + link copy)
    • 🎓 Instant PDF certificate download
    • 🙏 QR-based contribution support
    • ⏱️ 10-minute timer-based gameplay

🧩 Skill Areas Covered
    • Linguistic (language & expression)
    • Logical (math & reasoning)
    • Spatial (visual imagination)
    • Bodily-Kinesthetic (movement & coordination)
    • Musical (rhythm & sound)
    • Interpersonal (social intelligence)
    • Intrapersonal (self-awareness)

🎮 Game Flow
    1. Player starts the assessment
    2. Answers 70 quick questions
    3. Timer tracks engagement (10 minutes)
    4. Instant result generated:
        ◦ Skill scores (out of 30 each)
        ◦ Strength levels (Developing → Highly Dominant)
        ◦ Top 3 skills
        ◦ Career suggestions
    5. Player can:
        ◦ Share result 📤
        ◦ Download certificate 🎓
        ◦ Contribute via QR 🙏

🧠 Scoring Logic
    • Each skill = 10 questions
    • Each question = 0–3 points
    • Max per skill = 30
    • Total score → % based “Ultimate Profession Score”

💻 Tech Stack
    • HTML5
    • CSS3 (modern UI + gradients)
    • Vanilla JavaScript
    • jsPDF (for certificate generation)


🎯 Use Cases

    • Classroom engagement activity
    • Talent identification camps
    • Parent-child interactive learning
    • Educational fairs & competitions
